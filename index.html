Learning plan

Group main goal: Learn technical English using ISTQB materials.

Home task #1

Read about:

  • What is a Defect, error, failure (in general)
  • From where and when do defects arise
  • What is Testing types: Manual and Automation
  • What is a testing method: Static and Dynamic

The second lesson structure:

part #1 (1 hour)

  • We will work on new words vocabulary (correct pronunciation)
  • You can add your words that can include in our Fundamentals of Testing vocabulary
  • We will solve ISTQB questions that include the home task topics
  • You can add your questions from ISTQB exam for our home task topics

part #2 (1 hour)

  • Some of ours group members (who wanted) will introduce us to a short presentation about one of the home task topics
  • If some of our group members have questions about the material (difficult to understand, new word, etc.), we will try to help translate and answer all questions
